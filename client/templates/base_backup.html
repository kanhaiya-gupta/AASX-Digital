AASX Framework Help
What is AASX Digital Twin Analytics Framework?

The AASX Digital Twin Analytics Framework is a comprehensive platform for processing AASX (Asset Administration Shell) files and building digital twin analytics with ETL, Knowledge Graph, and AI/RAG capabilities.

How to use this framework:
  1. Start with AASX ETL Pipeline to process your AASX files
  2. Explore the Knowledge Graph to understand relationships
  3. Use the AI/RAG System for intelligent analysis
  4. Manage digital twins in the Twin Registry
  5. Handle compliance with Certificate Manager
  6. View insights in the Analytics Dashboard
Available Applications:
Core Processing
  • AASX ETL Pipeline: Extract, Transform, Load AASX files
  • Knowledge Graph: Neo4j-based relationship analysis
Analysis & Intelligence
  • AI/RAG System: AI-powered analysis and insights
  • Analytics Dashboard: Visualization and analytics
Management
  • Digital Twin Registry: Manage digital twin assets
  • Certificate Manager: Handle compliance and certification
Supported Infrastructure
  • Additive Manufacturing Facilities
  • Hydrogen Filling Stations
  • Smart Grid Substations
  • Wastewater Treatment Plants
  • And more critical infrastructure
Tips:
  • Upload AASX files through the ETL Pipeline first
  • Use the Knowledge Graph to explore relationships between assets
  • Ask natural language questions in the AI/RAG System
  • Monitor digital twin health in the Analytics Dashboard
About AASX Framework
Framework Overview

The AASX Digital Twin Analytics Framework provides a complete solution for processing AASX files and building digital twin analytics with ETL, Knowledge Graph, and AI/RAG capabilities.

Core Components
  • ETL Pipeline: Automated processing of AASX files
  • Knowledge Graph: Neo4j-based relationship analysis
  • AI/RAG System: OpenAI-powered intelligent analysis
  • Vector Database: Qdrant for semantic search
  • Digital Twin Registry: Asset management and monitoring
  • Certificate Manager: Compliance and certification
Technology Stack
  • Backend: FastAPI, Python 3.8+
  • Database: Neo4j, Qdrant
  • AI: OpenAI GPT-4.1-mini
  • Frontend: Bootstrap 5, JavaScript
  • Deployment: Docker, Docker Compose
