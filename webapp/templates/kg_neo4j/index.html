How to Use Neo4j
Docker Neo4j:
  • Quick setup with containerized instance
  • Runs on ports 7475 (browser) and 7688 (bolt)
  • Perfect for development and testing
  • Easy to start/stop with management controls
Local Neo4j Desktop:
  • Full desktop application with advanced features
  • Multiple database management
  • Advanced visualization and plugins
  • Professional development environment
Connect Neo4j Desktop to Docker:
  1. Launch Neo4j Desktop using the button above
  2. Create a new project or open existing one
  3. Click "Add Database" → "Connect to existing database"
  4. Use these connection details:
    • URI: bolt://localhost:7688
    • Username: neo4j
    • Password: password
  5. Test connection and save
